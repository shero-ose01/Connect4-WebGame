# Connect 4
A simple Multiplayer Connect 4. Create a Game/Room send link to another person and you can play together.

## Stack
ASP.NET for the backend, this time minimal Api, and Angular for the frontend. This is running on <a href="connect4.s-ose.de">connect4.s-ose.de</a> and is "self" hosted with nginx and Cloudflare tunnel.
